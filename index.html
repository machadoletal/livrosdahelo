⚙ Configurar GitHub

Para sincronizar automaticamente, você precisa:
1. Criar um repositório público no GitHub
2. Fazer upload do index.html e do books.json
3. Ativar o GitHub Pages em Settings → Pages → Deploy from branch
4. Criar um token em github.com/settings/tokens com escopo repo
